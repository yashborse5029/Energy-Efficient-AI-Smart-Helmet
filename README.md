## 📖 Project Overview

AI Smart Helmet is an intelligent embedded safety system designed to improve motorcycle rider safety by combining accident detection, emergency communication, voice interaction, GPS tracking, and real-time rider assistance into a single wearable device.

The system continuously monitors the rider's movement using an MPU6050 Accelerometer and Gyroscope. If a severe accident is detected, the helmet automatically starts an emergency response sequence. Before sending alerts, the system asks the rider:

> **"Are you okay?"**

The rider is given a short time to cancel the emergency if they are safe. If no response is received, the helmet automatically transmits the rider's live GPS location via GSM and initiates emergency communication.

Built on Raspberry Pi 4, the system integrates multiple embedded hardware modules while maintaining low power consumption, making it suitable for future smart transportation and intelligent safety applications.
